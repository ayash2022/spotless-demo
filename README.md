# spotless-demo

- This is a demo project to show how to include spotless in your SpringBoot project with Maven build tool. 
- Spotless helps in maintaining coding format uniform also enforces verification during maven execution phase.
- This helps during codereview and code walkthrough and avoi unnecessary space and formatting issue.
- You can use below command to apply spotless in mvn , if not used after code change then during compilation phase it build will fails.
- `mvn spoless:apply`
